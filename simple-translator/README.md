# Simple Translator

## Instructions
This is a program that can translate English to selected languages (currently adding more languages). Simaply type a word and select the language, the translated word will be display in the output section. Only meaningful words will be translated, otherwise English characters will be shown. 

## Setup

### Make sure you have the latest git, Node.js and Npm installed on your machine
```
$ git --version
$ npm --version
$ node --version
```
### Clone the repo from Github
`git clone https://github.com/LEO0331/simple-translator.git`

### Install dependencies and Run prettier
`npm install` & `npx prettier . --write`

### Start the project
#### `npm start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
The page will reload when you make changes.\
You may also see any lint errors in the console.

#### `npm test`
Launches the test runner in the interactive watch mode.\

#### `npm run build`
Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.
The build is minified and the filenames include the hashes.\
Your app is ready to be deployed!
